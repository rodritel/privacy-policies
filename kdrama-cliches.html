Privacy Policy

KDrama Cliches - Twitch Extension

Introduction

This privacy policy describes how the "KDrama Cliches" Twitch extension ("the Extension") handles user data. The Extension is developed by Rodritel.

Data Collection

The Extension does not collect, store, or transmit any personal data. It does not use cookies, analytics, tracking pixels, or any other data collection mechanism.

How the Extension Works

The Extension displays a static list of KDrama cliches as an overlay on the video player. All content is rendered locally in the viewer's browser. No data is sent to any external server.

Third-Party Services

The Extension does not integrate with any third-party services, APIs, or external databases. The only external dependency is the Twitch Extension Helper provided by Twitch.

User Accounts

The Extension does not require users to create an account or log in. It does not access any Twitch user data such as usernames, email addresses, or viewing history.

Children's Privacy

The Extension does not knowingly collect any data from children or any other users.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date below.

Contact
